User asked whether artifacts are abstracted for general ML training/eval/test/inference projects vs. tied to autoresearch (the v1 proof target).

**Verdict: well-abstracted; autoresearch coupling is contained.**

| `skills/llm-training-eval/SKILL.md` | 0 | Fully abstract — schema, metrics, tolerance bands generalize to any pre-training run |

| `commands/check-experiment.md` | 0 | Fully abstract |
| `agents/verifier.md` Phase 3a | 0 | Activation conditions are signal-based (`train.py` exists, `torch`/`jax`/`tensorflow` deps, `program.md` exists), not autoresearch-specific |

| ADR drafts | Mentioned in `dec-draft-bc906d09` Context, `dec-draft-7170187d` Context | ADR Context sections correctly cite autoresearch as the *empirical observation that prompted* the decision; the **Decision** sections are general |

The single mild concern: `skills/ml-training/SKILL.md` line 15 says "Activate alongside agentic-sdks and agent-evals for dual-archetype autoresearch" — this binds activation guidance to one project. Suggested rewording: "Activate alongside agentic-sdks and agent-evals for projects combining ML training with agentic loops (autoresearch is the v1 reference)." Filed as Suggested S1.

The artifacts handle **fine-tuning, inference-evaluation, and post-training** without modification because:
- `val_bpb` is the *primary* pre-training metric but the schema's `metrics:` block accepts `eval_harness: {}` for arbitrary harness output and `train_loss_final` for any training shape
- Compute-budget rules are agnostic to training paradigm
- Tiered backend strategy (local + SkyPilot + opt-in adapters) generalizes to any GPU workload
- `program.md` is opt-in by file presence; non-experiment-loop projects simply don't have one

**Suggested follow-up beyond this audit's scope:** consider a small `## Beyond Pre-training` callout in `skills/ml-training/SKILL.md` mapping the same artifacts to fine-tuning (frozen base + adapter weights) and inference-eval (no training loop, just `lm-evaluation-harness` runs). Not merge-blocking; an opportunity for follow-up clarity.
